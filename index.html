What is null to ?

Simply put, Null to Node is an experiment. In order to understand what that experiment is, you need understand what we mean when we say null and node. Null is a computer science concept meaning nothing. Literally nothing. It has no context, no meaning, and is different from zero. node.js is a compilation of Google's V8 JavaScript engine, Joyent's libuv platform layer, and a core library. In other, simpler words, node.js is JavaScript run the server.

Putting those two concepts together, Null to Node is having no knowledge whatsoever of node.js to being a proficient and budding web developer. Null to Node is a free web development course being taught by Brian Holt and Thom Allen in Salt Lake City aiming to going from ground zero to being able to get a job doing web development.
