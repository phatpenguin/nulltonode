Learn to code. For free.

Null to node is a free web development course being taught in Salt Lake City. Classes will take place every Tuesday at 8pm starting May 28th. No experience is required, in fact, if you have substantial experience this class may not be for you. Location and other information coming soon. Contact us on twitter if you have any questions.

So what is null? Null is a computer science concept meaning nothing. Literally nothing. It's a keyword in the Javascript language that represents emptiness, and is different from zero. Node.js is a compilation of Google's V8 JavaScript engine, Joyent's libuv platform layer, and a core library. In other, simpler words, node.js is JavaScript run on the server.

Putting those two concepts together, Null to Node is having no knowledge whatsoever of node.js to being a proficient and budding web developer. Our goal is to take you from a technical tabula rasa, to building functional real world web applications.
